# College Project (واجهة امامية فقط)

## الصفحات

- صفحة رئيسية لتسجيل الدخول عن طريق الرقم الدراسي
- صفحة عرض النائج
- صفحة حجب النتيجة لعدم دفع الرسوم
- صفحة تنبيه عند ادخال رقم دراسي خطأ

## المميزات

- تصميم عصري
- مناسب لجميع احجام الشاشات
- تفعيل صفحة طباعة النتيجة وحفظها ك مستند نصي
- انميشن جميلة للناجحين

## التقنيات المستخدمة

- HTML & CSS
- JavaScript

## ملاحظات

- لا تقم بتغيير ترتيب استدعاء الملفات
- لا تقم بتغيير  ترتيب العناصر داخل القالب
- لا تقم  بأخذ اكواد جافا سكريبت وتضعها بملف منفصل 
- قمت بوضع تعليق بجانب القيم الافتراضية التي يجب عليك تغييرها بالقيم التي سوف تجلبها من قاعدة البيانات
